# Support

## Where to Ask for Help

- Open a GitHub issue for bugs and feature requests
- Use Discussions (if enabled) for usage questions

## Before Opening an Issue

- Confirm you are on the latest beta
- Include macOS version and quickbox version
- Provide exact reproduction steps

## Diagnostic Bundle

For runtime issues, include:

- `quickbox` app version
- `quickbox` build number
- Relevant logs (without personal task content)
