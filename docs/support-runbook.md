# Support Runbook

## Common User Reports

1. Hotkey not working
2. Storage folder access errors
3. App not opening after update

## Triage Steps

- Confirm app version and macOS version
- Reproduce in latest beta build
- Ask for diagnostics log path only (no task content)

## Storage Access Recovery

1. Open Settings
2. Re-select storage folder
3. Retry capture

## Escalation

Escalate security-sensitive reports through private advisory only.
